💾 Cargar datos

Subí los Excel para actualizar el dashboard. Los datos se guardan en el navegador (IndexedDB).

📦 Datos de Picking
Archivo: Productividad arg.xlsx
Pestaña: "Ola"
Columnas: RESPONSABLE, FECHA, UNIDADES, LINEAS, TIPO
◯ Sin datos cargados
⚙️ Datos de Maquinistas
Archivo: productividad.xlsx
Pestaña: "Maquinista"
Columnas: FECHA, USUARIO, TOTAL_MOV, TOTAL_UND, tareas...
◯ Sin datos cargados
📝 Instrucciones de uso
Cómo actualizar el dashboard
1. Seleccioná el archivo Excel con el selector de archivo.
2. Presioná Cargar para procesar e importar los datos.
3. Los datos se guardan en IndexedDB del navegador — no se suben a ningún servidor.
4. Cada vez que cargás un archivo nuevo, los datos anteriores se reemplazan completamente.
5. Los datos persisten entre sesiones. Usá Borrar datos para limpiar.
